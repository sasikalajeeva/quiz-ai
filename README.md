# 🎓 Quiz-AI - Interactive Quiz Application

[![Deploy to GitHub Pages](https://github.com/sasikalajeeva/quiz-ai/actions/workflows/deploy.yml/badge.svg)](https://github.com/sasikalajeeva/quiz-ai/actions)

An interactive, feature-rich quiz application built with HTML, CSS, and JavaScript. Perfect for students!

## 🚀 Live Demo
[Play Quiz Now](https://sasikalajeeva.github.io/quiz-ai/)

## 📋 Features
- ✅ 6 Quiz Categories (Maths, Science, English, G.K., EVS, Mixed)
- ⏱️ 15-second timer per question
- 🎯 Real-time score tracking
- 🎉 Confetti celebration on good scores
- 📊 Progress indicators
- 🏆 Achievement badges
- 📱 Responsive design

## 🛠️ Tech Stack
- HTML5
- CSS3 (Custom animations)
- Vanilla JavaScript (No dependencies)
- GitHub Pages (Hosting)
- GitHub Actions (CI/CD)
